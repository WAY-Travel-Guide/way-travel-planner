OrangeButton

OrangeButton — компонент кнопки с предустановленными оранжевыми стилями (OrangeButton.css). Используется для действий по клику в различных формах, модальных окнах, панелях и т.д. Позволяет настраивать внешний вид с помощью prop style и поддерживает любой вложенный JSX через children.
Description:
