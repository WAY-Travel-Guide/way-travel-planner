PictureSwapper

PictureSwapper — React-компонент, реализующий слайдер (carousel) с изображениями, подписями и описаниями. Предназначен для визуального оформления страниц и привлечения внимания пользователя. Слайды пролистываются стрелками, плавно анимируются при смене. В верхней части есть быстрые ссылки: "Отзывы", "Контакты", "Помощь".
