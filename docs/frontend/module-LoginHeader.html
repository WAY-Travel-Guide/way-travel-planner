LoginHeader

LoginHeader — React-компонент, отображающий две навигационные кнопки для перехода на страницы логина и регистрации. Использует кастомный компонент Button (оранжевая кнопка) из UI-библиотеки приложения. Используется, например, в HomeWidget на главной странице.
