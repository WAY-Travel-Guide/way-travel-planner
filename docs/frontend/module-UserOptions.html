UserOptions

UserOptions — React-компонент для отображения опций пользователя в выпадающем меню. Содержит кнопки для перехода в профиль, просмотра истории и выхода из аккаунта. Кнопка "Выйти" возвращает на главную страницу и может быть расширена для очистки сессии.
