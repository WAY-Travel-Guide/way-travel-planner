StartForm

StartForm — React-компонент с приветственным текстом, коротким описанием сервиса и кнопкой "Приступим". Используется на странице пользователя для мотивации к началу работы с маршрутом. Подключает кастомные кнопки и ссылку "На главную" через ToHomeLogoButton.
