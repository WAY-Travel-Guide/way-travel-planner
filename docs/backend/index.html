user-module/middleware/roleMiddleware.js

Description:
  • - Проверяет наличие заголовка Authorization: Bearer . - Проверяет валидность JWT (подпись, срок действия). - Проверяет, что у пользователя в JWT есть хотя бы одна из указанных ролей. - Если проверки не пройдены, возвращает 403 Forbidden и сообщение. - Если всё ок, добавляет req.user с данными из JWT и пускает дальше.
